# 政策红利抓取工具

基于"钱"与"试点"双信号的政策红利调研分析系统

## 功能特性

- 📊 **数据可视化** - 实时图表展示政策资金流向
- 📝 **表单录入** - 支持手动更新各项政策数据
- 📁 **文件导入** - 支持JSON数据批量导入
- 📄 **报告生成** - 一键生成约20页完整DOCX调研报告
- 🔄 **历史追踪** - 自动保存数据变更历史

## 核心方法论

### 看钱 - 资金流向追踪
1. **部委预算拨款** - 官网搜索专项资金
2. **地方专项债** - 增长量=执行力度
3. **一级市场融资** - IT桔子等平台数据

### 看试点 - 政策先行区
1. **发改委试点名单** - 重点关注KPI达成情况
2. **超预期=行业拐点信号**

## 快速开始

### 方式一：本地运行

```bash
# 克隆仓库
git clone https://github.com/Joins1994/policy-dividend-tracker.git
cd policy-dividend-tracker

# 安装依赖
npm install

# 启动服务
node server.js

# 浏览器打开 http://localhost:3000
```

### 方式二：下载使用

直接下载仓库，解压后运行 `npm install && node server.js`

## 系统要求

- Node.js 14+
- 现代浏览器（Chrome/Edge/Firefox/Safari）

## 报告内容

生成的DOCX报告包含：
- 封面与更新说明
- 核心方法论框架
- 财政预算与专项债分析
- 一级市场融资信号
- 试点项目KPI追踪
- 四大重点领域深度分析
- 30分钟月度跟踪实操指南
- 风险提示与投资建议

## 数据来源

- 2026年政府工作报告
- 财政部预算草案报告
- 国家发改委政策文件
- IT桔子创投数据
- PitchBook AI投资数据

## 使用建议

1. **每月30分钟跟踪**：专项债流向、试点进展、融资数据
2. **每季度复盘**：预算变化、KPI达成、行业拐点
3. **抓住时间窗口**：政策落地前的最佳投资时机
